gulp-boilerplate.js

Success, you've found the sample page!

This boilerplate includes:

  • css vendor prefixing by autoprefixer (via gulp-autoprefixer)
  • css compression by csso (via gulp-csso)
  • js validation by jshint (via gulp-jshint)
  • js browserify bundling by watchify
  • js compression by uglify-js
  • image compression by image-min (via gulp-imagemin)
  • svg compression by svgo (via gulp-svgmin)
  • static http server by node-static
  • livereload server by gulp-livereload and tiny-lr

Available build tasks:

  • gulp — runs gulp validate and gulp build tasks
  • gulp validate — validates and lints all non-module javascript
  • gulp build — builds, minifies, and copies all client-side js/css/html from client/ to build/
  • gulp watch — same as gulp build, but watches files for changes and automatically rebuilds
  • gulp serve — runs gulp build, a simple static HTTP server, and a livereload server
    • Warning! This server is just for development use, not production.

Livereload notes:

To use livereload you'll need a javascript snippet in your page or a browser extension. Read more here.
